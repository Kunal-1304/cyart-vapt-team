# VAPT Lab Report – Full Cycle Assessment

## Overview
This project demonstrates a complete Vulnerability Assessment and Penetration Testing (VAPT) cycle, including theoretical understanding and practical implementation. The report covers advanced exploitation techniques, web application security testing, reporting practices, post-exploitation, and a final capstone project.

## Contents

### Section 1 – Theoretical
- Advanced Vulnerability Exploitation  
- Web Application Penetration Testing  
- Reporting and Stakeholder Communication  

### Section 2 – Practical
- Exploit chaining (XSS to RCE)
- Web application testing using Burp Suite and sqlmap
- Security reporting and visualization
- Post-exploitation and evidence collection
- Full VAPT cycle (Capstone Project)

## Tools Used
- Metasploit Framework  
- Burp Suite  
- sqlmap  
- OpenVAS (Greenbone)  
- Wireshark  
- Nmap  
- DVWA  

## Key Outcomes
- Successfully chained vulnerabilities to achieve remote code execution  
- Identified SQL Injection and XSS vulnerabilities  
- Performed privilege escalation and gained root access  
- Captured and validated evidence using hashing  
- Conducted a full VAPT cycle following PTES methodology
