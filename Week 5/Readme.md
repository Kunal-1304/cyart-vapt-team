# Week 4 – Advanced VAPT

## Theoretical Knowledge

Advanced Exploitation covers exploit chaining, custom PoC development, and bypassing protections like ASLR, DEP, and WAF.  

API Security Testing focuses on OWASP API Top 10, using tools like Burp Suite and Postman to identify issues such as BOLA and injection flaws.  

Privilege Escalation and Persistence includes exploiting SUID, kernel vulnerabilities, and maintaining access using cron jobs and system tools.  

Network Protocol Attacks involve SMB relay, Man-in-the-Middle attacks like ARP spoofing and DNS poisoning, and identifying misconfigurations.  

Mobile Application Testing includes OWASP Mobile Top 10, static analysis with MobSF, and dynamic testing using Frida.  

Reporting and Remediation focuses on CVSS scoring, PTES-based reports, and clear communication with stakeholders.  

---

## Practical Labs

The Exploitation Lab involves chaining exploits, creating custom PoCs, and bypassing defenses using ROP techniques.  

The API Testing Lab includes testing endpoints, exploiting BOLA and GraphQL injection, and documenting findings.  

The Privilege Escalation Lab focuses on enumeration with LinPEAS, gaining root access, and setting up persistence.  

The Network Attacks Lab includes SMB relay attacks, ARP spoofing, and analyzing traffic using tools like Wireshark.  

The Mobile Testing Lab involves APK analysis, identifying insecure storage, and bypassing security using Frida.  

The Capstone Project requires performing a full VAPT on a target machine, including exploitation, reporting, and remediation.
